HTML

Hyper Text Markup Language

HTML is de basis voor het schrijven van een webpagina. Hiermee breng je structuur aan in je webpagina.

HTML elementen

  1. h1

    2. titel

  2. p

    3. paragraaf

  3. head

    4. Hier vind je alle informatie over de pagina. (metadata)

  4. body

    5. de daadwerkelijke pagina (dit verschijnt op je scherm)

  5. hr

    6. horizontal ruler: een horizontale lijn.

  6. title

    7. Titel die in de tab komt te staan.

Links en Afbeeldingen

  1. a

    2. linkelement. Hierin staat de tekst waar je de link onder wil zetten.

  2. href

    3. attribuut voor de link

  3. target

    4. attribuut waar de link moet openen. met target _blank opent de pagina in een nieuw tabblad.

  4. img

    5. element voor afbeeldingen

  5. src

    6. source attribuut. Als je wilt linken naar een afbeelding.

  6. width

    7. breedte van een afbeelding.

  7. div

    8. groepeert een gedeelte van de website voor de opmaak van de website.

Semantic HTML

  1. header

    2. Navigatie en logo

  2. nav

    3. navigatie elementen

  3. main

    4. Content: artikelen, foto's etc.

  4. article

    5. los onderdeel wat op zihzelf staat

  5. section

    6. sectie van de website

  6. titels

    7. H1 is de titel van de website. h2, h3, h4 etc. zitten in de secties van de website.

  7. aside

    8. niet gelinkt aan de inhoud, bijvoorbeeld een advertentie.

  8. Footer

    9. Navigatie en social media.
    Kan ook in een article gebruikt worden voor bijvoorbeeld copyright.
